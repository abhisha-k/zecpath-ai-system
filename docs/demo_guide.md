# Demo Guide

## Objective

This document explains how to demonstrate the AI-powered Applicant Tracking System using sample resumes and job descriptions.

---

## Demo Steps

1. Select a sample resume.
2. Upload the resume to the ATS.
3. Parse resume content.
4. Extract sections, skills, experience, and education.
5. Perform semantic matching with the job description.
6. Generate ATS score.
7. Rank the candidate.
8. Produce recruitment reports.
