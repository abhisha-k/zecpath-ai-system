# Accuracy Metrics

## 1. Objective

The Accuracy Metrics document evaluates the performance of the AI-powered Applicant Tracking System (ATS) by measuring how accurately it classifies, scores, and ranks candidates. These metrics help assess the reliability and effectiveness of the recruitment pipeline.

---

## 2. Module Overview

The ATS evaluation compares the predicted recruitment decisions generated by the system with the expected outcomes for different candidate profiles. The comparison is used to calculate performance metrics such as overall accuracy and success rate.

---

## 3. Evaluation Workflow

```
Sample Candidate Profiles
          │
          ▼
 ATS Prediction Generation
          │
          ▼
Compare with Expected Results
          │
          ▼
Performance Evaluation
          │
          ▼
Accuracy Metrics Report
```

---

## 4. Evaluation Dataset

| Parameter | Description |
|-----------|-------------|
| Test Dataset | Sample candidate profiles |
| Total Test Cases | 4 |
| Candidate Types | Fresher, Experienced, Senior |
| Job Categories | Technical and Non-Technical |
| Evaluation Method | Expected vs Actual Result Comparison |

---

## 5. Evaluation Metrics

| Metric | Description |
|--------|-------------|
| Total Test Cases | Total resumes evaluated |
| Correct Predictions | Predictions matching expected results |
| Incorrect Predictions | Predictions differing from expected results |
| Overall Accuracy | Percentage of correct predictions |
| Success Rate | Overall system performance |

---

## 6. Metric Calculation

The overall accuracy is calculated using the following formula:

```
Accuracy (%) =
(Number of Correct Predictions / Total Test Cases) × 100
```

For the current implementation:

| Metric | Value |
|---------|------:|
| Total Test Cases | 4 |
| Correct Predictions | 4 |
| Incorrect Predictions | 0 |
| Overall Accuracy | 100% |
| Success Rate | 100% |

> **Note:** Update these values if your future testing produces different results.
