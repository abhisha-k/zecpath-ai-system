# Storage Structure

## Objective

This document defines how AI recruitment data is stored in the Zecpath platform.

## Folder Structure

storage/

- resumes/
- parsed_profiles/
- ats_scores/
- screening_reports/
- interview_results/
- metadata/
- model_versions/

## Data Formats

| Data | Format |
|------|--------|
| Resume | PDF / DOCX |
| Candidate Profile | JSON |
| ATS Score | JSON |
| Screening Report | PDF / JSON |
| Interview Result | JSON |
| Metadata | JSON |

## Benefits

- Organized storage
- Easy retrieval
- Scalable architecture
- Supports AI workflows
